# Herndon

Herndon is an agent-focused Python CLI for creating Excel workbooks from structured JSON specs.

It is built for deterministic, local-first workbook generation:

- create and mutate sheet specs through the CLI
- validate workbooks or individual sheets
- inspect normalized workbook state as JSON
- render `.xlsx` output without GUI automation

See [SPEC.md](SPEC.md) for the full v1 specification.
